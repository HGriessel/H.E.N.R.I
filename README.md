## Usage of <command> <flags>:
  -c string
        the type of note defaults to Inbox
                - i = Inbox
                - p = Project
                - a = Area
                - r=Resource
                - z=archive (default "i")
  -h string
        must specify title -t and sub-category -s type of category will default to inbox or specify  
                - i = Inbox
                - p = Project
                - a = Area
                - r=Resource
                - z=archive  (default "i")
  -s string
        name of SubCatergory
  -t string
        title of note
